## expressions conditionelles et instruction if else

Commençons avec les expressions conditionnelles et l'instruction if else. Une 
expression conditionnelle est une expression qui a pour résultat un booléen
c'est-à-dire True ou False. C'est toujours le résultat d'un test. Les expressions
conditionelles les plus courantes sont [a > b], [a >= b], [a < b], [a <= b], [a == b], 
et [a != b]. Et on peut également rajouter le test d'appartenance des séquences
[a in s] et [a not in s]. 

Les expressions conditionnelles sont principalement utilisées
dans une expression if else. L'instruction if va permettre d'executer un bloc d'instruction
si l'expression est vraie est un autre bloc si l'expression est fausse. 

Regardons a cas pratique : supposons que l'on veuille écrire un programme 
qui affiche reçu si la note d'un examen est supérieur à 10 et recalé sinon.
